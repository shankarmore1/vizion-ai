# ✨ Vizion AI

### From Raw Data to Real Insights — Your AI Business Analyst

Vizion AI is an AI-powered business analytics assistant built entirely on AWS serverless architecture. It combines conversational AI with image understanding today — and will analyze spreadsheets, visualize data, and answer questions about your business tomorrow.

## 💡 Why Vizion?

Most AI chatbots are general-purpose. Vizion is different — it's being built specifically for **business analytics workflows**. Whether you need to understand a chart, analyze a dataset, or get quick insights from your data, Vizion is your AI-powered analyst that works 24/7.

**Today:** Conversational AI with image understanding
**Tomorrow:** Excel analysis, data visualization, and project-specific knowledge bases

## 🚀 What Vizion Can Do Today

- 💬 **Smart Conversations** — Context-aware chat that remembers your conversation
- 🖼️ **Image Analysis** — Upload charts, dashboards, screenshots — Vizion reads and explains them
- 🌙 **Dark / Light Theme** — Clean, professional UI for long working sessions
- 📱 **Works Everywhere** — Desktop, tablet, and mobile ready
- ⚡ **Streaming Responses** — Watch answers appear in real-time
- 📥 **Export Chat** — Save your analysis conversations for later
- 🛡️ **Secure** — Protected API with authentication, throttling, and monitoring

## 🏗️ How It Works

> **User** → **Amplify** (Frontend) → **API Gateway** → **Lambda** (Backend) → **Bedrock** (AI)

| Step | What Happens |
|------|-------------|
| 1 | User sends a message or uploads an image (chart, dashboard, etc.) |
| 2 | Frontend sends authenticated request to the API |
| 3 | Backend validates the request and loads conversation context |
| 4 | Amazon Bedrock AI processes the request |
| 5 | Conversation history is saved for context memory |
| 6 | Uploaded images are securely stored |
| 7 | AI response streams back to the user in real-time |

## 🛠️ Built With

| Layer | Technology |
|-------|-----------|
| **Frontend** | HTML5, CSS3, Vanilla JavaScript |
| **Backend** | Python 3.12, AWS Lambda |
| **AI** | Amazon Bedrock (text + vision models) |
| **API** | Amazon API Gateway |
| **Database** | Amazon DynamoDB |
| **Storage** | Amazon S3 |
| **Hosting** | AWS Amplify |

## 🔒 Security

- Authenticated API access with secret key
- Request throttling and rate limiting
- Input validation and sanitization
- Automated cost monitoring and budget alerts
- Session-based access control
- CORS protection
